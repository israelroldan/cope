# Microsoft Power BI

✅ Approved · **Category:** Analytics & BI · **AI Maturity:** high · **Confidence:** high · **Agent:** Copilot

## Summary

A business intelligence platform that connects to various data sources to visualize data, generate reports, and share insights across an organization.

## Quick Facts

- **Company:** Microsoft
- **Website:** https://powerbi.microsoft.com
- **Deployment:** hybrid
- **Data Residency:** Global (configurable via Microsoft Fabric/Azure regions)
- **Compliance:** GDPR, HIPAA, FedRAMP, ISO 27001, SOC 1/2/3
- **Agent:** Copilot ([Source](https://powerbi.microsoft.com/en-us/blog/the-next-era-of-copilot-in-power-bi-chat-with-your-data/))

## Score Rationales

- **AI Maturity:** Features deep native integration of Generative AI (Copilot) and traditional ML capabilities for automated insights.
- **Confidence:** Extensively documented enterprise product with clear official roadmaps and feature lists.

## AI Readiness Rationale

- Includes Copilot for creating reports via natural language prompts.
- Offers 'AI Insights' for sentiment analysis and key phrase extraction.
- Integrates seamlessly with Azure Machine Learning models.

## Landscape Tags

**Roles:** Analytics
**CIB Domains:** Insights and Analytics, Customers and Contracts, Projects and Innovation

- Primary tool for transforming raw data into actionable visual insights.
- Serves as the centralized presentation layer for organizational metrics.

## Key Features

- Interactive Dashboards
- Data Connectivity (Connectors)
- Data Transformation (Power Query)
- Paginated Reports
- Mobile App Access

## AI Highlights

- Copilot in Power BI for report generation
- Natural language Q&A for data querying
- Smart Narratives for automated text summaries
- Key Influencers visualization
- AutoML integration via Azure

## Integrations

- Microsoft 365 (Excel, Teams, PowerPoint)
- Azure Data Services
- Salesforce
- Google Analytics
- SQL Server

## Specifications

- DAX (Data Analysis Expressions)
- M Formula Language
- Power BI Desktop (Windows only)
- Fabric Capacity (for Copilot)

## Risks & Governance

- Data leakage through improperly shared dashboards
- Reliance on accurate underlying data models (GIGO)
- Cost management for Premium/Fabric capacities
- Shadow IT via uncontrolled workspace creation
