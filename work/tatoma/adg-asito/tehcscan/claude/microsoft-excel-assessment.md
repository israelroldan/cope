# Microsoft Excel

✅ Approved · **Category:** Analytics & BI · **AI Maturity:** high · **Confidence:** high · **Agent:** Agent Mode

## Summary

A spreadsheet application used for organizing, formatting, and calculating data with formulas.

## Quick Facts

- **Company:** Microsoft
- **Website:** https://www.microsoft.com/excel
- **Deployment:** hybrid
- **Data Residency:** Global (dependent on Microsoft 365 tenant configuration)
- **Compliance:** SOC 2, ISO 27001, GDPR, HIPAA (via Microsoft 365 Trust Center)
- **Agent:** Agent Mode ([Source](https://www.microsoft.com/en-us/microsoft-365/blog/2025/09/29/vibe-working-introducing-agent-mode-and-office-agent-in-microsoft-365-copilot/))

## Score Rationales

- **AI Maturity:** Deep integration with Microsoft Copilot allows for natural language data analysis, formula generation, and visualization.
- **Confidence:** Ubiquitous enterprise tool with extensive official documentation and widespread adoption.

## AI Readiness Rationale

- Copilot can generate formulas, charts, and insights from natural language prompts.
- Analyze Data feature automatically identifies trends, patterns, and outliers.
- Integration with Python allows for advanced machine learning workflows directly within the grid.
- AI-driven features like 'Insert Data from Picture' digitize analog information.

## Landscape Tags

**Roles:** Analytics, System Of Record, Workflow
**CIB Domains:** Insights and Analytics, Projects and Innovation, Customers and Contracts

- Primary tool for ad-hoc data analysis and visualization across all business functions.
- Acts as a system of record for small datasets, financial models, and departmental trackers.
- Facilitates workflows through macros, VBA automation, and Power Automate integrations.

- Spreadsheet grid interface
- PivotTables and PivotCharts

- VBA and Macros
- Conditional Formatting
- Real-time collaboration (Excel for Web)

## Specifications

- Supports millions of rows via Data Model
- Available on Windows, macOS, Web, iOS, Android
- Uses .XLSX open XML format
- Supports Python integration

## Risks & Governance

- Shadow IT risk due to local file storage and lack of central visibility
- Version control challenges with shared files
- Data accuracy risks from manual entry errors or broken formulas
- Macro-based security vulnerabilities
