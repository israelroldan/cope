Error Handling
🚫
No AI Features Found
Document clearly in the report. Shift focus to the third-party integration section as the primary value.
🔒
Paywalled Content
Note the limitation in the report. Focus on publicly available information and free review platforms.
🔄
Conflicting Information
Priority order: official docs > recent reviews > older content. Note conflicts in the report.
⏲
Rate Limiting
Pause 30 seconds between searches if the browser encounters errors from search engines.
Research Tips
Verify AI Name
Vendors rename AI features frequently. Always search for the current branded name.
GA vs Beta
Note if AI features are generally available or still in preview/beta.
Regional Availability
Some AI features are US/EU only. Flag this when found.
Pricing Tiers
Note if AI requires specific subscription levels (e.g., Enterprise only).
Recency Matters
AI evolves fast. Prioritize sources from the last 12 months.
