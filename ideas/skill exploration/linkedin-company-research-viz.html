LinkedIn Company Research

Automated research workflow that gathers company posts, updates, and news from LinkedIn and outputs a structured spreadsheet with clickable links.

Step-by-Step Workflow
1
Clarify Research Parameters
User Input
Ask the user what to research: company names, optional topic focus, and time range.
Click to expand
Company Name(s)
"HubSpot", "Salesforce", "Notion"
Topic Focus (optional)
"AI features", "product launches", "hiring"
Time Range
Default: past month
2
Navigate LinkedIn Search
Browser
Use LinkedIn's content search with company + topic keywords. Search is preferred over scrolling company pages for speed and filtering.
Click to expand
// Constructed search URL
linkedin.com/search/results/content/
  ?keywords=HubSpot%20AI
  &datePosted=%22past-month%22
  &origin=FACETED_SEARCH
Filter: Date
Past 24h / Week / Month
Filter: Content Type
Posts, Articles, Documents
Filter: From
Specific people or companies
3
Extract Post Data & URLs
JavaScript
For each search result, extract author, date, title, content preview, engagement metrics, and the direct post URL using JS injection.
Click to expand
// Batch-extract post URLs from search results
document.querySelectorAll(
  'a[href*="/posts/"], a[href*="/pulse/"], a[href*="/feed/update/"]'
).forEach(link => {
  console.log(link.href);
});
Collected Fields
Author, Date, Title, Content (200 chars), Engagement, URL
Target Volume
10-20 relevant posts per company
4
Generate Spreadsheet
Python
Use the bundled Python script (openpyxl) to create a formatted .xlsx file with styled headers, clickable hyperlinks, and auto-sized columns.

Error Handling
🔍
No Results
Try broader keywords or remove the topic filter to get a wider set of results.
🔒
Not Logged In
Prompt user to log into LinkedIn in their connected browser before continuing.
⏲
Rate Limiting
Pause 30 seconds between searches if LinkedIn returns errors.
